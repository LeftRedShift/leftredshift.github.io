ferramenta.run_attack_from_spec(spec: modules.registry.AttackSpec, resolved_params: Dict[str, Any], capture_enabled: bool = True, max_runtime_s: int = 15) → Dict[str, Any][source]¶

Função de execução dos ataques (controle de containers)

  • Inicia o container do ataque (docker run -d –rm …)

  • (Opcional) Inicia captura tcpdump em docker0 e encerra automaticamente quando o container terminar

  • Encerra o container após max_runtime_s se ainda estiver em execução (watchdog)

  • Registra logs em logs/attacks.log (best-effort)

Parameters:

  • spec (AttackSpec) – Difinição dos parâmetros do ataque vindos do registry

  • resolved_params (Dict[str, Any]) – Lista de parâmetros para a execução

  • capture_enabled (bool, optional) – Booleano para ativar automaticamente ou não a captura junto, padrão é True

  • max_runtime_s (int) – Tempo máximo (s) para encerrar o container caso ainda esteja rodando

Returns:

Dicionário de parâmetros

Return type:

Dict[str, Any]
